## Prerequisites
- Node.js >= 18 (if you pick Node.js) or Python 3.10+ (if you pick Python microservices)
- MongoDB or PostgreSQL running (this doc assumes **MongoDB** by default)
- Telegram Bot token (create via @BotFather)
- Optional: Redis (for rate-limits and active poison timers)
- Recommended: `pm2` or Docker for production process management
